## 本章小結

本章介紹了公有云服務對 Docker 的積極支援，以及新出現的容器雲平臺。

事實上，Docker 技術的出現自身就極大推動了雲端計算行業的發展。

透過整合公有云的虛擬機器和 Docker 方式，可能獲得更多的好處，包括

* 更快速的持續交付和部署能力；
* 利用核心級虛擬化，對公有云中伺服器資源進行更加高效地利用；
* 利用公有云和 Docker 的特性更加方便的遷移和擴充套件應用。

同時，容器將作為與虛擬機器類似的業務直接提供給使用者使用，極大的豐富了應用開發和部署的情境。
